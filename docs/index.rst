SLACM documentation
===================

SLACM, a 'Simple Lightweight Actor Model', was created to support the development a distributed, component-based, 
soft real-time applications in Python3. It is an application framework the provides a component model, as well as
component scheduling and messaging services. It is based on ZeroMQ and the threading and multi-processing capabilities 
available in Python.
